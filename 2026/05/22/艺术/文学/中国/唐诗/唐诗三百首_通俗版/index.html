145|## 哥舒歌 - 写英雄
146|
147|西鄙人〔唐代〕
148|
149|北斗七星高，哥舒夜带刀。
150|
151|至今窥牧马，不敢过临洮。
152|
153|1、晚上的天贼干净，北斗七星挂得老高老高，亮得晃眼，一眼就能瞅见。
154|
155|（注：不是单纯写星星，是说 “夜黑风高” 的晚上，适合巡逻站岗）
156|
157|2、咱边境的守护神哥舒翰老哥，大半夜的也不睡觉，腰上挎着明晃晃的大刀，在边境线上溜达巡逻，贼精神！
158|
159|（注：带刀不是装样子，是真刀真枪守着，谁敢来犯就砍谁）
160|
161|3、到现在那些北边的胡人（游牧民族），还贼心不死，总想偷偷摸摸跑过来抢马、抢粮食。
162|
163|（注：窥牧马 = 说好听点是偷看马群，其实就是 “趁人不备搞偷袭” 的黑话）
164|
165|4、但他们就算胆子再肥，也不敢跨过临洮这道坎儿 —— 因为哥舒翰在这儿守着呢！

167|## 静夜思 - 写思乡
168|
169|李白〔唐代〕
170|
171|床前明月光，疑是地上霜。
172|
173|举头望明月，低头思故乡。
174|
175|1、大半夜的，床跟前洒了一地白晃晃的月光，亮得晃眼睛。
176|
177|（注：这里的 “床” 不是咱们现在睡觉的软床，大概率是古代的井栏或者坐具，反正就是屋子跟前儿的一块地方）
178|
179|2、我迷迷糊糊睁开眼，还以为是天冷了，地上结了一层白霜呢！
180|
181|（注：疑 = 以为、看错了；霜是冬天地上的白碴子，这里就是说月光太亮太凉，跟霜一个样）
182|
183|3、揉了揉眼睛抬头一瞅，哦，原来是天上那轮大月亮在发光。
184|
185|4、瞅着瞅着月亮，我又低下头，心里一下子就酸了 —— 想家了，想老家的爹妈、老家的炕头了！
186|
187|> 这个意思是：失眠！！！

189|## 怨情 - 写闺怨
190|
191|李白〔唐代〕
192|
193|美人卷珠帘，深坐蹙蛾眉。(蹙 一作：颦)
194|
195|但见泪痕湿，不知心恨谁。
196|
197|1、一个长得贼带劲的妹子，伸手一把撩开了屋里挂着的珠帘。
198|
199|（注：卷珠帘 = 不是慢慢卷，是有点不耐烦地撩开；珠帘 = 珠子串起来的帘子，看着挺精致，说明妹子家境不差）
200|
201|2、然后她蔫蔫地往屋里一坐，一屁股怼在那儿，皱着两道好看的眉毛，愁眉苦脸的，谁都不想搭理。
202|
203|（注：深坐 = 往屋子深处坐，躲着人；
204|
205|蹙蛾眉 = 皱眉头，不是生气，是心里堵得慌，皱得眉头都快拧成疙瘩了）
206|
207|3、你瞅过去就能看见，她脸上还挂着湿乎乎的泪痕，眼泪都把脸蛋儿浸得湿漉漉的了。
208|
209|（注：但见 = 就看见；泪痕湿 = 不是刚哭完，是哭了好一会儿，眼泪没干，还挂着呢）
210|
211|4、大白话：就是不知道她心里头到底憋着啥气，恨的是哪个王八蛋，把自己委屈成这样！

218|

228|

230|
231|4、可瞅着瞅着就叹气了 —— 可惜啊可惜！这么好看的夕阳，马上就要天黑了，没一会儿就得没了！

236|
237|李端〔唐代〕
238|
239|鸣筝金粟柱，素手玉房前。
240|
241|欲得周郎顾，时时误拂弦。
242|
243|—-
244|
245|那架古筝的琴柱上，还镶着金灿灿的小珠子，看着就贼精致，一弹起来叮叮当当响得好听。
246|
247|（注：金粟柱 = 古筝上用来固定琴弦的柱子，镶了金粟似的装饰，不是真的金子做的；鸣筝 = 古筝响起来）
248|
249|弹古筝的姑娘，伸出那双白嫩白嫩的小手，坐在自家精致漂亮的屋子跟前儿，正准备开弹呢。
250|
251|（注：素手 = 姑娘白嫩的手；玉房 = 不是玉石做的房子，就是形容屋子华丽讲究）
252|
253|这姑娘心里头憋着个小九九 —— 就盼着那个懂音乐的帅哥（跟周瑜似的）能多看自己一眼。
254|
255|（注：周郎 = 周瑜，三国大佬，超懂音律，谁弹错曲子他就会回头瞅一眼，后来 “周郎顾” 就成了 “让心上人注意自己” 的梗）
256|
257|所以啊，她故意不好好弹，老是假装手滑，时不时就弹错个音，为的就是让那帅哥能回头看她一眼！

261|宋之问〔唐代〕
262|
263|岭外音书断，经冬复历春。
264|
265|近乡情更怯，不敢问来人。

270|
271|（注：岭外 = 岭南，唐朝时的偏远蛮荒之地，被贬的人才会去；音书断 = 没信儿、没消息）

278|
279|（注：怯 = 不是害怕路上有贼，是怕家里出事儿，怕一问就是坏消息）
280|
281|路上碰见从老家那边过来的人，嗓子眼都痒痒想打听，可话到嘴边又咽回去了

296|
297|（注：三分国 = 魏蜀吴三足鼎立；盖 = 压过、超过，就是说诸葛亮帮刘备打天下的功劳，没人能比）
298|

300|
301|（注：八阵图 = 诸葛亮摆的一种军用阵法，用石头堆的，能困住敌人的千军万马，贼拉厉害）
302|
303|哪怕江水天天哗哗地冲，那些摆阵的石头愣是没被冲挪窝，稳得一批！
304|
305|（注：江流 = 长江的水；石不转 = 石头不动，既写实景，又说诸葛亮的本事和志向，跟石头一样硬）
306|
307|可他这辈子最大的遗憾，就是没能拦住刘备非要去打吴国的糊涂账，最后蜀汉也跟着栽了！
308|
309|（注：失吞吴 = 刘备执意讨伐吴国，结果大败，蜀汉元气大伤，诸葛亮想匡扶汉室的心愿也泡汤了）

311|## 宿建德江 - 写羁旅愁思
312|
313|孟浩然〔唐代〕
314|
315|移舟泊烟渚，日暮客愁新。
316|
317|野旷天低树，江清月近人。
318|
319|把船撑到一片雾蒙蒙的小水洲边上，停下锚，今晚就搁这儿过夜了。
320|
321|（注：烟渚 = 雾气腾腾的小沙洲，不是啥正经码头，就是个荒郊野外的临时停靠点）
322|
323|眼瞅着天就黑透了，我一个在外漂泊的人，心里头又冒出来一股子新的愁绪，堵得慌。
324|
325|（注：客 = 在外的旅人，就是孟浩然自己；愁新 = 不是旧愁没消，又添了新愁，愁上加愁）
326|
327|往四周一瞅，这野外空旷得没边儿，天看着比树尖儿都低，压得人喘不过气来。
328|
329|（注：野旷 = 荒野辽阔，一眼望不到头；天低树 = 不是天真的变低了，是空旷的地方看着天离树更近，显得压抑）
330|
331|再低头看脚下的江水，清得能看见水底的沙子，天上的月亮映在水里，好像离我就几步远，陪着我似的。

738|

762|
763|这可不是真的问眉毛！朱庆馀写这首诗，是要拿去给考官张籍（张水部）看的。他把自己比作新媳妇，把张籍比作老公，把科举考试比作拜见公婆。这话的潜台词是：“我写的文章咋样？合不合您老人家的胃口？能不能考上啊？”
